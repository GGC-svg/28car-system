# 28Car 車輛管理系統

香港 28car 二手車資訊自動收集、管理、分析系統。

---

## 快速安裝（給一般用戶）

### 步驟 1：解壓縮
將整個資料夾放到你想要的位置（例如：`D:\28Car`）

### 步驟 2：執行安裝
**右鍵點擊「一鍵部署安裝.bat」→ 以系統管理員身分執行**

安裝程式會自動：
- 建立桌面捷徑
- 設定開機自動啟動
- 設定每日自動更新資料（早上 6:00）
- 設定每日自動備份（凌晨 5:00）
- 設定防火牆（允許區域網路連線）
- 啟動伺服器

### 步驟 3：開始使用
- 雙擊桌面「28Car 開啟網頁」
- 或在瀏覽器輸入：http://localhost:5000

## 其他裝置連線

在同一個 WiFi 網路下，手機或其他電腦可以連線使用：

1. 在主電腦查看 IP 位址（安裝完成時會顯示）
2. 在其他裝置瀏覽器輸入：`http://主電腦IP:5000`

例如：`http://192.168.1.100:5000`

## 功能說明

### 自動功能
- **每日資料更新**：每天早上 6:00 自動從 28car 抓取最新資料
- **每日資料備份**：每天凌晨 5:00 自動備份資料庫

### 手動功能（管理後台）
- 可在「管理後台」→「排程管理」修改自動執行時間
- 可手動立即執行更新或備份

### 主要功能
- 🚗 車輛列表：瀏覽所有車輛，支援多種篩選和排序
- 👥 聯絡人目錄：自動整理賣家資訊，識別車行/同行/私人
- 📝 溝通紀錄：記錄與賣家的聯繫情況
- 📊 同款查價：查看同款車型的價格參考
- 📱 簡訊功能：批量發送簡訊（需設定簡訊服務）

## 常見問題

### Q: 安裝時出現「需要管理員權限」
A: 右鍵點擊安裝檔，選擇「以系統管理員身分執行」

### Q: 防火牆阻擋連線
A: 執行安裝程式時會自動設定，若仍有問題請手動允許 TCP 5000 埠

### Q: 資料如何備份？
A: 系統每天凌晨 5:00 自動備份到 `backup` 資料夾

### Q: 如何完全移除？
1. 刪除桌面捷徑
2. 刪除啟動資料夾中的捷徑（`%APPDATA%\Microsoft\Windows\Start Menu\Programs\Startup`）
3. 刪除排程任務（執行 `schtasks /delete /tn "28car_daily" /f` 和 `schtasks /delete /tn "28car_backup" /f`）
4. 刪除整個程式資料夾
