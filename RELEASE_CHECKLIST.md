# 28Car 發佈打包檢查清單

## 打包步驟

### 1. 執行打包（開發者電腦）
```
雙擊 build_exe.bat
```
這會產生 `release` 資料夾，包含所有發佈檔案。

### 2. 發佈資料夾結構
```
release/
├── 28car_server.exe      # 主程式（必要）
├── 28car_scraper.exe     # 爬蟲程式（必要）
├── index.html            # 網頁介面（必要）
├── 一鍵部署安裝.bat       # 安裝腳本（必要）
├── run_daily.bat         # 每日排程（必要）
├── backup_db.bat         # 備份腳本（必要）
├── README.md             # 使用說明（建議）
├── VERSION               # 版本資訊（建議）
└── static/               # 靜態資源（必要）
    └── no-image.svg
```

### 3. 用戶安裝流程
1. 解壓縮到任意位置（如 `D:\28Car`）
2. 右鍵「一鍵部署安裝.bat」→ 以系統管理員身分執行
3. 安裝完成，開始使用

## 自動功能確認

| 功能 | 執行時間 | 說明 |
|------|----------|------|
| 開機自動啟動 | 電腦開機時 | 捷徑放入啟動資料夾 |
| 每日資料備份 | 05:00 | Windows 排程任務 |
| 每日資料更新 | 06:00 | Windows 排程任務 |

---

## 測試項目

- [ ] 一鍵安裝執行正常
- [ ] 伺服器可正常啟動
- [ ] 網頁可正常開啟 (http://localhost:5000)
- [ ] 登入功能正常（admin/admin）
- [ ] 排程任務已建立（檢查工作排程器）
- [ ] 防火牆規則已建立
- [ ] 桌面捷徑可用
- [ ] 開機自動啟動有效

---

## 注意事項

1. **管理員權限**：安裝時需要管理員權限才能設定排程和防火牆
2. **防毒軟體**：某些防毒軟體可能誤報，需加入例外
3. **首次登入**：預設帳號 admin，首次登入會要求更改密碼
